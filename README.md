Indie Studio is an Epitech bomberman project made in Irrlicht.

To play on Linux distro :
~ mkdir build
~ cd build
~ cmake ..
~ make
~ ./bomberman

To play on Windows 64 :
Launch executable in /out/build/x64-Release
